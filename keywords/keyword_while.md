# Keyword while

## Descrption

Declares a while loop. Used to execute a statement, or statements, repeatedly.

## Example

```cpp
int startIndex = 0;
const int TARGET_INDEX = 10;

while startIndex < TARGET_INDEX;
    println "Looping... index is: ", startIndex;
end;
```
