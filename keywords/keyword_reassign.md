# Keyword reassign

## Description

Reassign a variable to a different variable.

## Example

```cpp
if type == "string";
	string newArray doubleSize;
	int i = 0;
	while i < size;
		newArray[i] = standard_vector[i];
		i += 1;
	end;
	reassign standard_vector newArray;
end;
```

```standard_vector``` will now point to ```newArray```.
