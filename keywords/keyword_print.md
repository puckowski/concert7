# Keyword print

## Descrption

Print a string to standard out.

## Example

```cpp
string WELCOME_MESSAGE = "Welcome!";

print WELCOME_MESSAGE;

string token1 = "This";
string token2 = "test print.";

print token1, " is a ", token2, "\n";
```
