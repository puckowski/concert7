# Keyword instanceof

## Description

Returns the type of a variable.

## Example

```cpp
mutex lock;
string varType;

# varType will be set to "mutex"
instanceof lock varType;

# varType will be set to "int"
int testInt = 5;
instanceof testInt varType;
```
